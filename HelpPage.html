SCHEDULE DATA TRANSFER

WHAT IS THE SCHEDULE DATA TRANSFER (SDT)?

Schedule Data Transfer (SDT) is a web app that allows you to automatically download a Google Sheet from Google Drive to a local server in Seagate (server has to support SFTP/FTPS) according to your specified schedule.

Any Seagator with Seagate’s Gmail and Google Drive account can use this app. For first time users, this app will need the approval of the user to access his/her Google Drive files for it to download.

HOW DO I ACCESS THE APPLICATION?

You can access the app via the ‘freckles’ menu (the 3x3 boxes) from your Seagate Gmail/Drive/Calendar (You’ll have to scroll/click on the ‘More’ link if you can’t find it). Look for the Scheduled Data Transfer icon.

Note:
First time users to the app will be forwarded to a Google Approval page, where you’ll be requested to provide the app access to your Google Drive file for it to pick and download to your target location. Click on the Accept button to proceed.

HOW DO I SCHEDULE THE GOOGLE SHEET DOWNLOAD?

Step 1 : Click on the New Download button. The Schedule Download Job form will appear.

Step 2 : Fill in the following form details;

Job Description

Fill in the schedule name with an easy to identify name that let’s you differentiate and know what the scheduled job is for. You can add more descriptions for this job in the Description text field.

Spreadsheet to Download

Click on the Choose File… button to select your Google Sheet from your Drive. A dialog box containing your Google Sheets will appear when clicked.

You can search for your Google Sheets from the search box. It will search again all Sheets you own/those shared with you. Double-click on the file to select it and the dialog box will close.

You’ll notice that the URL of the Google Sheet you selected has been populated on the read-only text field. Notice the new [View] link to the right of the text field. You can click on this link to open and verify the Sheet you have selected on a new tab/window.

Below the text field, there’s a dropdown list titled [Download As:]. Here, you’ll be able to select what file you’d like to download the Google Sheet as. The options available are to download as CSV (Comma Separated Value), MS Excel (.xlsx), and Open Office Spreadsheet (.ods).

Also give special attention to the [(Optional) Target file name:] text field below;

This text field allows you to modify the downloaded file name to a name of your choosing. If nothing has been selected, then the system will download it using the Google Sheet’s file name.

Target Download Server Information

This is where you define where you want the system to drop your Google Sheet file (within Seagate of course).
The server where you want to download the file to has to support SFTP or FTPS (for Tableau).
You’ll need the server hostname, which folder on the server you want to download it too, and the username and password account owned by you to that server.
Fill in all the details, and select which [Server Protocol] you’re using to connect to that server.

You can click on the Test Connection button to find out if the server details you entered is correct and if our app can connect to that server.

Download Schedule

To schedule the automatic download according to your time, select among the timezone available in the dropdown (based on Seagate’s site for convenience).

To specify a specific interval for your schedule, select the Scheduling Type dropdown, and the appropriate text fields/checkboxes/dropdowns will appear depending on your preferred scheduling type.

Step 3 : Click on the Schedule New Job button. That’s it!

HOW DO IS SORT JOBS ON THE JOB LIST TABLE ?

In the main Download Jobs list, you’ll see a table with several of your jobs. By default all the jobs are sorted based on the latest created jobs on top. Each of the table column can be sorted. Click on the column header that you would like to do the sorting, either in ascending or descending order.

HOW TO FILTER/SEARCH THE JOB FROM JOB LIST TABLE ?

Step 1 : Navigate to the Schedule Download Spreadsheet > List screen and the Job List table will display the jobs that created by you and the list is sorted by [Created On] column in descending sequence by default.

Step 2 : The system allow you to do the filtering by each and every columns. You can filter/search the job by entering the keyword in the column/columns that you wish to filter/search on.

The system allow you to do the filtering by each and every columns. You can filter/search the job by entering the keyword in the column/columns that you wish to filter/search on.

HOW TO PAUSE AND RESUME A JOB ?

Step 1 : Search or Filter the job that you would like to pause/resume by following the 'How to search and view the jobs created' steps.

Step 2 : You will be leading to this page once you click the selected Job name. Click on the 'Pause Job' button in order to pause the job.

Step 3 : Click on the Resume Job button in order to resume the job.

HOW TO EDIT THE JOB DETAILS ?

Step 1 : Search or Filter the job that you would like to pause/resume by following the 'How to search and view the jobs created' steps.

Step 2 : Click on the Edit button to edit the job details.

Step 3 : Edit the existing download job with the information that you would like to change. Click on the Save button once you have done the editing.

HOW DOES THE FAILED JOB RETRY MECHANISM WORKS?

There are 2 types of Job retry in place for this application to run more effectively :

  1. Job failed for the first time regardless of any exception, the system will attempt to trigger only one time of job retry after 5 mins.
  2. Job failed for the first time and the failure is due to ‘Read timeout’, the system will trigger 3 times of job retries with first download waiting time is 20 seconds, 2nd time is 2 minutes and 3rd time is 5 minutes.

WHAT IS THE FILE SIZE LIMITATION?

Old version of Google Spreadsheets: 400,000 cells, with a maximum of 256 columns per sheet. Uploaded spreadsheet files that are converted to the Google spreadsheets format can’t be larger than 20 MB, and need to be under 400,000 cells and 256 columns per sheet.

New version of Google Sheets: 2 million cells, with a maximum of 256 columns per sheet. Uploaded spreadsheet files that are converted to the Google spreadsheets format can’t be larger than 100MB, and need to be under 2 million cells and 256 columns per sheet.

Please note that extremely large spreadsheets may have slower performance.
